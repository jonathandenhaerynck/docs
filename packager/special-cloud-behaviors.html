Special cloud behaviors

info

This page is about the TurboWarp Packager.

This option changes the behavior of specifically named cloud variables to unlock new compatibilities for your project. This is based on a similar feature in HTMLifier.

To make these, just create a cloud variable as you normally would but give them the specific name found below. For example, to use the â˜ url variable, create a variable named url and mark it as a cloud variable.

Enabling special cloud behaviors will override any other settings for these variables, so a variable like â˜ username will never be stored locally or synced with other users.
